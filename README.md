# homebrew-kener-ctl

Homebrew tap for [kener-ctl](https://github.com/supporterino/kener-ctl) — a declarative CLI for managing [Kener](https://kener.ing) status pages.

## Install

```bash
brew tap supporterino/kener-ctl
brew install kener-ctl
```

## Upgrade

```bash
brew upgrade kener-ctl
```

## Uninstall

```bash
brew uninstall kener-ctl
brew untap supporterino/kener-ctl
```
